# Front-Mobile

튜터와 학생 간의 과외 매칭 및 관리를 위한 모바일 애플리케이션입니다.

## 📌 프로젝트 소개
Front-Mobile은 튜터와 학생이 서로 매칭하고
과외 일정 관리, 채팅, 수업 진행 등을 원활하게 할 수 있도록 돕는 모바일 앱입니다.

## 🚀 주요 기능

- 카카오 소셜 로그인
- 튜터 / 학생 역할 선택
- 튜터 - 학생 매칭
- 과외 일정 관리 (캘린더)
- 채팅 기능
- 알림 기능

## 🛠 기술 스택

- **Framework** : Flutter
- **State Management** : Riverpod
- **Architecture** : Clean Architecture
- **API Communication** : REST API
